# AnySearch
AnySearch может сделать поиск по выбранной вами системе поиска.
## Поддерживаемые системы
- Google
- Yandex
- Bing
- DuckDuckGo
- Yahoo
- Рамблер
- VK (Mail.ru)
- Baidu
- You.com
- Ask
- еще другие...
